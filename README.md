# Predicting Plant Growth Stages Using Power BI

## Project Overview
This project analyzes plant growth data using Power BI to identify optimal environmental and management conditions for plant growth.

## Tools Used
- Power BI Desktop
- DAX
- CSV Dataset

## Dataset
Plant Growth Data Classification dataset from Kaggle.

## Features
- Interactive dashboard
- KPI cards
- Filters (slicers)
- Growth milestone prediction

## Files Included
- PowerBI Dashboard (.pbix)
- PowerBI Dashboard (.pdf)
- Dataset (.csv)
- Project Report (.pdf)

## Insights
- Loam soil shows highest growth milestones
- Daily watering improves growth percentage
- Moderate temperature and humid conditions optimize growth
